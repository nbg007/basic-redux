# Basic Example

This is a quick example for React + Redux. You can use it to play around. 

# What's In Da Box?!

- Babel: ES6 + Stage 0 ES7
- LiveReload: Webpack + Hot Module Replacement for
- Reux DevTools
- Sensible ESLint defaults

- `npm i`
- `npm run start`
- Then, open http://localhost:3001
